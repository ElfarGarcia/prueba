clonar proyecto: git clone y luego pegar direcion de repositorio
mostrar archivos nuevos no subidos: git status
subir el archivo a la parte local: git add y nombre archivo 
para revisar la diferencia de contenido entre el archivo existe en la parte local y el que va a subir:
git diff
subir el archivo :git commit o git commit -m"mensaje que quiera para identificar los cambios realizados"
subir los cambios a git:git push
